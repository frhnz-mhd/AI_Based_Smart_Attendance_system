# AI-Based Smart Attendance System Using Face Recognition Technology

## Project Overview

This project is an AI-Based Smart Attendance System that automates student attendance using face recognition technology. The system allows administrators to register students, capture face images, train a face recognition model, and automatically mark attendance through facial recognition.

## Technologies Used

* Flutter (Mobile Application)
* Flask (Backend API)
* SQLite (Database)
* OpenCV (Face Detection & Recognition)
* LBPH Face Recognition Algorithm
* HTML, CSS, JavaScript

## Features

### Admin Features

* Admin Login
* Student Registration
* Face Image Capture
* Face Recognition Model Training
* Attendance Records Management
* CSV Export
* Dashboard Analytics

### Student Features

* Student Login
* Face Recognition Attendance
* Attendance History
* Notifications
* Profile Management

## AI Features

* Face Detection
* Face Recognition using LBPH
* Automatic Attendance Marking
* Image-Based Student Identification

## Database Tables

* users
* students
* attendance

## How to Run the Project

### Backend

```bash
pip install -r requirements.txt

```

### Flutter Application

```bash
flutter pub get
```
