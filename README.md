# AI-Based Smart Attendance System Using Face Recognition Technology

## Project Overview

The AI-Based Smart Attendance System is a mobile and web-based attendance management solution that automates student attendance using Face Recognition Technology. The system eliminates manual attendance marking, reduces proxy attendance, and provides accurate attendance records in real time.

The project consists of an Admin Module and a Student Module. Administrators can register students, capture face images, train the face recognition model, manage attendance records, and export reports. Students can log in, mark attendance using face recognition, view attendance records, and receive notifications.

---

## Objectives

* Automate student attendance marking using face recognition.
* Reduce manual attendance management.
* Prevent proxy attendance.
* Improve attendance accuracy and efficiency.
* Provide attendance reports and analytics.

## Project Workflow

1. Admin registers a student.
2. Student face images are captured.
3. The face recognition model is trained.
4. Student logs into the mobile application.
5. Student marks attendance using face recognition.
6. Attendance data is stored in the database.
7. Admin views attendance records and reports.

## Installation Guide

### Backend Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Flask server:

```bash
python app.py
```

---

### Flutter Setup

Install Flutter dependencies:

```bash
flutter pub get
```

Run application:

```bash
flutter run
```

Build APK:

```bash
flutter build apk --release
```
